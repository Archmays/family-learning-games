# Stop/Help 动作小路

## 游戏目标

孩子在生活场景中练习 `stop`、`help`、`no`、`turn` 四个安全和轮流动作词。

## 适合对象

适合 3-5 岁，适合亲子共玩。家长读场景，孩子可以先做动作，再点英文单词。

## 玩法说明

完整题库有 500 个场景站点，每轮随机抽取 10 站。每站给出成人口令、动作提示和 inline SVG 内置插图，例如红灯、走丢、陌生人糖果、热杯子、朋友摔倒、排队等待和轮流玩玩具。孩子选择对应英文词，答题后会看到中文场景化反馈。

## 涉及知识点

学科为英语，也包含生活规则和动作反应。英文保持 RAZ aa 难度：单词、短短语和成人口头输入，不要求独立英文阅读。内容为通用安全场景，不写入孩子姓名或真实家庭观察。

## 设备适配

支持鼠标和触控。按钮大、文字短，适合手机、平板和电脑。可使用浏览器语音朗读英文单词，但不依赖音频完成游戏。

## 当前完成度

可玩。完整题库已扩到 500 站，每次仍只玩 10 站短轮次，重玩会重新抽取内容。现在支持完整一轮、完成总结、重新开始和返回大厅。

## 后续改进建议

后续可增加更多真实生活小场景，并记录是由动作提示、场景提示还是延迟回忆触发正确反应。

## 接入方式

导出 `actionPathGame`，在 `packages/data/gameCatalog.ts` 中注册。
