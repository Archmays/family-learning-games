# 汉字大转盘

## 游戏目标

通过内外部件组合，观察汉字和词语如何构成，并练习读音、例词和构词理解。

## 适合对象

- 年龄：6-15 岁。
- 适合亲子共玩：家长可以带读拼音、解释例词，并引导孩子造句。

## 玩法说明

选择年级和模式后，转盘随机组合两个部件。孩子观察组合结果，读出汉字或词语，并查看拼音和例词。

## 涉及知识点

- 汉字部件组合。
- 拼音和例词。
- 小学到初中分层汉字材料，数据来自 `packages/data/learningGames.ts`。

## 设备适配

- 支持鼠标和触控。
- 适合手机、平板和电脑。
- 不需要键盘、音频或打印材料。

## 当前完成度

可玩。已接入大厅，支持年级分层、字模式和词模式。
