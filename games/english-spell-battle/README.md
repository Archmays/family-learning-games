# 英文魔法战

## 游戏目标

通过图像、发音和拼写练习常见英语词汇。

## 适合对象

- 年龄：5-9 岁。
- 适合亲子共玩：家长可以先读中文或英文，孩子选择或拼出单词。

## 玩法说明

孩子先在图鉴中预习词义和发音，再完成首字母、乱序拼写和完整拼写三种挑战。

## 涉及知识点

- 基础英语词汇。
- 首字母识别。
- 单词拼写。
- 词库位于 `packages/data/learningGames.ts`。

## 设备适配

- 支持鼠标和触控。
- 适合手机、平板和电脑。
- 发音使用浏览器语音能力，可能受设备设置影响。

## 当前完成度

可玩。已接入大厅，包含图鉴预习、分级挑战和本地进度记录。

## 后续改进建议
