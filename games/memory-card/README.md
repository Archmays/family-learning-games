# 记忆翻牌

## 游戏目标

通过翻牌配对练习观察、短时记忆和汉字字形辨认。

## 适合对象

- 年龄：4-8 岁。
- 适合亲子共玩：家长可以提示孩子观察字形、偏旁或图像线索。

## 玩法说明

每次翻开两张卡片。如果两张属于同一组，就保持翻开；如果不匹配，会短暂展示后盖回。目标是在较少步数内找完所有配对。

## 涉及知识点

- 汉字字形观察。
- 记忆配对。
- 年级分层汉字材料，数据来自 `packages/data/memoryCards.ts`。

## 设备适配

- 支持鼠标和触控。
- 适合手机、平板和电脑。
- 不需要键盘、音频或打印材料。

## 当前完成度

可玩原型。已经接入大厅和本地存档，会记录不同年级题组的最佳步数和完成次数。

## 后续改进建议

- 增加更明确的完成反馈和亲子提示语。
- 为低龄儿童增加更大的卡片模式。

## 接入方式

- 导出：`memoryCardGame`。
- 注册位置：`packages/data/gameCatalog.ts`。
