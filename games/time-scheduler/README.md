# 时间任务调度员

## 游戏目标

孩子练习把日常任务放到合适时间，并把经过时间拆成小时和分钟来理解。

## 适合对象

适合 6-8 岁，尤其适合已经开始学习时间安排、冲突判断和简单经过时间的孩子。建议家长让孩子说出选择理由。

## 玩法说明

当前有 6 个日常任务和 3 个经过时间题。游戏先给出整理书包、读短句、饭前洗手、午后游戏、整理桌面、睡前准备等任务，孩子选择合适时间。完成安排后，进入经过时间挑战，先跳整小时或整十分钟，再计算剩下分钟。

## 涉及知识点

学科为数学，结合生活管理。知识点包括时间先后、任务冲突、合理安排和经过时间拆分。内容使用通用家庭日程，不写入真实学习记录或家庭观察原话。

## 设备适配

支持鼠标和触控。按钮使用完整时间标签，时间轴和拆分条适合手机、平板和电脑竖屏。不需要键盘、音频或横屏。

## 当前完成度

可玩。已从 3 个任务 + 1 个经过时间题扩展为 6 个任务 + 3 个经过时间题，并加入 Level 1 inline SVG 任务贴纸和时间拆分条。现在支持完整一轮、完成总结、重新开始和返回大厅。

## 后续改进建议

后续可增加可拖动时间轴、更多真实日程，以及跨半小时/跨整点的经过时间题。

## 接入方式

导出 `timeSchedulerGame`，在 `packages/data/gameCatalog.ts` 中注册。
