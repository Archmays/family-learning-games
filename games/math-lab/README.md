# 数学实验室

这是现有 Phaser 数学实验室游戏的大厅接入层。

当前阶段不移动 `src/game/` 中的实现，避免破坏资源加载、测试路径和场景依赖。后续如果需要独立演进，再规划一次小步迁移。

## 当前产品原则

- 面向 6-9 岁儿童，保持短回合、低压力、可解释。
- 苹果园巡逻、甜点屋练习、小河靠岸、图书馆挑战、集市配订单随时可玩。
- 星星、贴纸、徽章只记录练习历史和掌握证据，不作为进入资格。
- `masteryGate` 只用于准备度建议和复练建议，不阻止孩子选择任意环节。
- 不引入联网、登录、广告、支付、排行榜、倒计时、抽卡或强惩罚。

## 掌握与复练

- 题目继续手写维护，每题需要技能标签、三层提示、三段反馈和可解答案。
- 存档按技能记录独立完成、提示后完成、失误后完成。
- 家长摘要展示技能状态、提示依赖、常错方向和复练建议，但不做儿童画像。
- 1/3/7 天复练节奏只在游戏内显示建议，不做通知或强制任务。

## 奖励与可访问性

- 奖励页合并阶段贴纸和成就徽章，文案绑定“练会了什么”。
- 未获得奖励显示温和目标，不显示失败感或锁定感。
- 设置支持大字、减少动画、高对比度和左手布局。
- 后续新增视觉素材前，优先用代码内贴纸和现有 UI 风格验证玩法价值。
