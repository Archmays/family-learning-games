# 数学实验室

## 游戏目标

在苹果园、甜点屋、小河、图书馆和集市场景中练习加减法、数量关系和简单两步推理。

## 适合对象

- 年龄：6-9 岁。
- 适合亲子共玩：家长可以陪孩子读题、复述数量关系，并观察孩子是否需要提示。

## 玩法说明

孩子选择一个数学场景，阅读任务目标，通过点击、选择或输入完成加减法任务。游戏会给出提示、反馈和阶段奖励。

## 涉及知识点

- 加法、减法和数量调整。
- 多步骤计划。
- 场景化问题理解。
- 内容数据位于 `public/data/levels/` 和 `src/game/domain/`。

## 设备适配

- 支持鼠标和触控。
- 适合手机、平板和电脑。
- 包含图片资源和音效反馈，运行资源位于 `public/assets/` 和 `public/data/`。
