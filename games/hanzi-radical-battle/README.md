# 汉字魔法战-偏旁部首

- 目标：通过偏旁部首组合认识汉字结构和常见字义。
- 适合年龄：6-10 岁。
- 学科方向：识字。
- 核心玩法：选择英雄，点击手牌中的偏旁部首，组成汉字后释放魔法攻击怪物。
- 难度系统：怪物随关卡提升血量，每 5 关出现更强怪物。
- 计分方式：本游戏按通关关卡、发现新字和战斗记录进行单局反馈，不持久保存成绩。
- 亲子互动：适合家长提示偏旁含义，孩子尝试找出能组成的汉字。
- 打印材料：不需要打印材料。
- 电脑交互：支持鼠标和触控，音效需要浏览器允许用户交互后播放。
- 接入大厅：导出 `hanziRadicalBattleGame`，在 `packages/data/gameCatalog.ts` 注册。
