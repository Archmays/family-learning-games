# 汉字魔法战-偏旁部首

## 游戏目标

通过偏旁部首组合认识汉字结构和常见字义，在战斗反馈中强化组合关系。

## 适合对象

- 年龄：6-10 岁。
- 适合亲子共玩：家长可以提示偏旁含义，孩子尝试找出能组成的汉字。

## 玩法说明

孩子选择英雄，在手牌中点击偏旁部首，组成有效汉字后释放魔法攻击怪物。战斗会随着关卡推进逐步提高难度。

## 涉及知识点

- 偏旁部首。
- 汉字结构和组合。
- 常见字义和词语联想。
- 游戏数据位于 `games/hanzi-radical-battle/game-data.ts`。

## 设备适配

- 支持鼠标和触控。
- 适合手机、平板和电脑。
- 音效需要浏览器允许用户交互后播放。
