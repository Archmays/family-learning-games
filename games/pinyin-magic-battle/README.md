# 汉字魔法战-拼音

## 游戏目标

练习常见汉字拼音、声调辨认和认读反应。

## 适合对象

- 年龄：6-8 岁。
- 适合亲子共玩：家长读汉字，孩子找出正确拼音。

## 玩法说明

孩子先通过图鉴认读汉字和拼音，再进入勇者试炼，选择正确拼音攻击怪兽。

## 涉及知识点

- 常见汉字拼音。
- 声调辨认。
- 汉字认读。
- 拼音卡片数据位于 `packages/data/learningGames.ts`。

## 设备适配

- 支持鼠标和触控。
- 适合手机、平板和电脑。
- 不需要键盘或打印材料。

## 当前完成度

可玩。已接入大厅，包含图鉴认读、拼音选择战斗和本地最佳成绩。

## 后续改进建议

- 增加按声调分类练习。
- 增加易混拼音复练。

## 接入方式

- 导出：`pinyinMagicBattleGame`。
- 注册位置：`packages/data/gameCatalog.ts`。
