# 生态证据侦探

## 游戏目标

孩子通过看图、读提示和点选答案，练习用证据解释简单生态关系，而不是只凭直觉猜答案。

## 适合对象

适合 6-8 岁，尤其适合已经能进行亲子图文讨论、愿意说理由的孩子。建议家长一起玩，先让孩子说图上的证据再点选。

## 玩法说明

当前有 8 道题。每题展示一个由 inline SVG 内置插图、箭头和证据提示组成的生态场景板。孩子选择答案后，游戏会提示应该看哪条证据。题目覆盖能量来源、谁被谁吃、分解者、生产者、叶片咬痕、栖息地和数量变化。

## 涉及知识点

学科为科学。知识点包括食物链、能量来源、生产者、分解者、捕食关系、栖息地和简单数量变化。内容使用通用家庭学习场景，不写入孩子姓名、真实学习记录或家庭观察原话。

## 设备适配

支持鼠标和触控。按钮较大，场景板在手机宽度下会自动变成两列，适合平板、电脑和手机竖屏。

## 当前完成度

可玩。已从 4 题扩到 8 题，并把原来的 emoji 字符串替换为 Level 1 inline SVG 内置插图。现在支持完整一轮、完成总结、重新开始和返回大厅。

## 后续改进建议

后续可增加更多经过审核的生态场景，并把“能量来自”和“被吃”两种箭头做成更明确的图例练习。

## 接入方式

导出 `ecologyEvidenceGame`，在 `packages/data/gameCatalog.ts` 中注册。
