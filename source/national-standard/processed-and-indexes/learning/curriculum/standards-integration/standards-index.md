# Standards Index

本目录用于把不同国家和地区的课程标准整合成家庭可用的综合学习地图。

## 已知来源

- 中国义务教育课程方案和课程标准 2022
- Singapore MOE syllabuses
- Common Core State Standards
- NGSS 和 California NGSS
- IB PYP 资料
- Australian Curriculum v9
- Canada BC Curriculum

## 使用方式

1. `standard-outcomes.jsonl` 保存标准条目的摘要级结构化结果。
2. `standard-crosswalk.csv` 把来源标准映射到家庭版 canonical objectives。
3. `strengths-by-standard.md` 记录不同标准的优势和家庭可借鉴点。
4. `integration-principles.md` 说明整合原则，避免直接照搬任何一个体系。
