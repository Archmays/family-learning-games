# Curriculum Reference Files

These files are development references only. They are not loaded by the game at runtime.

- `chinese-curriculum-2022.pdf`: 教育部《义务教育语文课程标准（2022年版）》, downloaded from the official Ministry of Education attachment page.
- `math-curriculum-2022.pdf`: 教育部《义务教育数学课程标准（2022年版）》, downloaded from the official Ministry of Education attachment page.
- `english-curriculum-2022.pdf`: 教育部《义务教育英语课程标准（2022年版）》, downloaded from the official Ministry of Education attachment page.
- `general-standard-chinese-characters-2013.pdf`: 《通用规范汉字表》 reference for validating commonly used standard characters.

The current Hanzi wheel keeps the existing 1-3 grade prototype data. A full 1-6 grade replacement should wait for a verified textbook character-list research report with sources and page references.
